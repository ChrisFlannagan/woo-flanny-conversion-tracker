QTC for Woo is a way to quickly build links that include any code you can think of and will track conversions

== Description ==

This is the quickest and easiest way to track conversions from affiliates, newsletters and more.  Simply add a code to the end of any link to your site.

If you have a newsletter you are sending out you could add ?qtc_woo_tracking_code=Oct2016Newsletter to the end of every link from it.  Then in your admin type that code in and see the results at any time.

The plugin also supports google analytics enhanced ecommerce for tracking conversions in your google analytics panel.
